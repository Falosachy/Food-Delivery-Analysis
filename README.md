# Food Delivery Analytics

Analyze and visualize customer delivery performance using real‑world food delivery data. This repository includes a Jupyter notebook that:

1. **Data Ingestion & Cleaning**  
   - Loads the delivery dataset  
   - Handles missing values and outliers

2. **Exploratory Analysis**  
   - Distribution of delivery ratings (pie chart)  
   - Delivery duration vs. rating (box plot)  
   - Delivery area vs. rating heatmap

3. **Outlier Detection**  
   - Identifies anomalies using Z‑score and IQR methods

4. **Insights & Recommendations**  
   - Highlights patterns in on‑time performance  
   - Suggests operational improvements for faster, higher‑rated service
